# SpotifyApp

## About
In this project I've tried to make some Spotify functionality.

### Used Technologies: 
- [x] Spotify API
- [x] UIKit (w/out Storyboards)
- [x] SDWebImage
- [x] WebKit
- [x] AVFoundation
- [ ] Appirater
- [ ] Firebase

## Functionality:
- Fetch data through API (Albums, Artists, Tracks, Releases etc.)
- Sign In / Sign Out (OAuth 2.0)
- Play preview of tracks, albums, playlists.
